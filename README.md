# Personal website

Static website for GitHub Pages.

## Deploy with GitHub Pages

Use a repository named `USERNAME.github.io` for a personal site at:

`https://USERNAME.github.io/`

Alternatively, use any repository name and enable GitHub Pages from the `main` branch for a project site at:

`https://USERNAME.github.io/REPOSITORY/`

This site does not require Hugo, Jekyll, npm, or a build step.
